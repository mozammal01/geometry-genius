What is DOM and what is the use of DOM ?

DOM means document object model.It is a programming api for html.It defines the logical structure of documents and the way a document is accessed and manipulated.. It represents the page so that programs can change the document structure, style, and content.

How to select element from DOM ?

1.getElementByID
2.getElementsByClassName
3.getElementsByTagName
4.querySelector
5.querySelectorAll

What is Event Bubbling ?

HTML elements receive different types of events, from click, to blur, to scroll, and so on. One behavior these events have in common is Event Bubbling.

What is callback function and what is work of it ?

A callback's primary purpose is to execute code in response to an event.A JavaScript callback is a function which is to be executed after another function has finished execution.These events might be user-initiated, such as mouse clicks or typing. With a callback, you may instruct your application to "execute this code every time the user clicks a key on the keyboard." button.
